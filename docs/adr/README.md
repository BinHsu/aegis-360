# Architecture decision records

ADRs record decisions that constrain the project. They explain context,
decision, and consequences; design details and experimental results belong in
their respective documentation areas.

## Status meanings

- **Accepted**: current authority until explicitly superseded.
- **Proposed**: under review and not binding.
- **Superseded**: retained for history; follow the named replacement.
